Auth
====

.. req:: POST /authorize
   :id: REQ_INTEROP_086
   :status: verified
   :tags: Auth

   The endpoint shall perform an authorization flow and issue authorization information based on client credentials.

.. req:: POST /token
   :id: REQ_INTEROP_087
   :status: verified
   :tags: Auth

   The endpoint shall issue or refresh an access token that authorizes subsequent SOVD API calls.
