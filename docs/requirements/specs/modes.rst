Modes
=====

.. req:: GET /{entity}/modes
   :id: REQ_INTEROP_053
   :status: open
   :tags: Modes

   The endpoint shall list all supported modes of operation for the addressed entity.

.. req:: GET /{entity}/modes/{mode-id}
   :id: REQ_INTEROP_054
   :status: open
   :tags: Modes

   The endpoint shall return information about the addressed mode of operation.

.. req:: PUT /{entity}/modes/{mode-id}
   :id: REQ_INTEROP_055
   :status: open
   :tags: Modes

   The endpoint shall request a change of the entity into the addressed mode of operation.
